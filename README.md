## Sobre o Projeto

Modelo de **Regressão Linear** para prever a eficiência de combustível de um carro em **km/galão**, com base no peso do veículo. A partir de um conjunto de dados simples, o modelo aprende a relação entre peso e consumo e realiza previsões para novos valores.

> 💡 Projeto desenvolvido e executado no **Google Colab**.

---

## 📊 Dataset

| Peso (1.000 km) | km/galão |
|:-:|:-:|
| 3,50 | 18 |
| 3,69 | 15 |
| 3,44 | 18 |
| 3,43 | 16 |
| 4,34 | 15 |
| 4,42 | 14 |
| 2,37 | 24 |

---

## ⚙️ Como Funciona

```
Peso do carro (1.000 km) → Regressão Linear → km/galão previsto
```

| Etapa | Descrição |
|-------|-----------|
| 1. Importação | `numpy` para dados e `LinearRegression` de `sklearn` |
| 2. Preparação | Dados formatados para o padrão do scikit-learn |
| 3. Treinamento | Instância de `LinearRegression` treinada com os dados |
| 4. Previsão | Modelo prevê km/galão para novos pesos de entrada |
